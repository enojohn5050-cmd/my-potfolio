# 🌐 My Portfolio Website

Welcome to my personal portfolio website! This project was built using only **HTML** and **CSS** as part of my frontend development learning journey.

## 📖 About This Project

This is my first portfolio website. I created it to practice my web development skills, showcase my work, and provide a place where potential clients and employers can learn more about me and the services I offer.

## ✨ Features

The website includes:

* A hero section with a brief introduction
* A statistics section highlighting experience and achievements
* Service cards showcasing what I offer
* A client testimonial section
* A contact call-to-action for potential clients

## 🚀 How to View the Project

### Option 1: Open Locally

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` to launch the website in your browser.

Visit the live website here:
