# My Portfolio Website

A personal portfolio website I built using HTML and CSS.

## What is this?

This is my first portfolio website. I built it to show my web design skills and attract clients who need a website.

## What's on the page?

- A hero section with a short intro about what I do
- A stats section showing my experience
- Cards showing the services I offer
- A client testimonial
- A contact call-to-action at the bottom

## How to view it

**Option 1 — Open locally:**
Download the files, then double-click `index.html` to open it in your browser.

**Option 2 — View live:**
[Click here to see the live site](https://github.com/enojohn5050-cmd/my-potfolio)
